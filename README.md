# Machine Learning Models

This repository contains hands-on implementations of core machine learning concepts and algorithms using Python and scikit-learn. It includes regression, classification, clustering, dimensionality reduction, and model evaluation techniques applied to real-world datasets.

The goal of this repository is to build strong foundational knowledge in machine learning through structured labs, assignments, and a final end-to-end project.

---

## 📚 Topics Covered

### 🔹 Supervised Learning
- Linear Regression
- Multiple Linear Regression
- Polynomial Regression
- Logistic Regression
- Decision Trees
- K-Nearest Neighbors (KNN)
- Support Vector Machines (SVM)

### 🔹 Unsupervised Learning
- K-Means Clustering
- Hierarchical Clustering
- Principal Component Analysis (PCA)
- t-SNE
- UMAP

### 🔹 Model Optimization & Evaluation
- Train/Test Split
- Cross Validation
- Regularization (Ridge)
- Feature Scaling
- Pipeline Implementation
- Hyperparameter Tuning
- Performance Metrics (R², MSE, Accuracy, Confusion Matrix)
